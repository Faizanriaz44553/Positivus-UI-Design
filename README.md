# Positivus - Modern Agency Landing Page

A clean, modern, and fully responsive landing page UI inspired by the Positivus design. Built from scratch using semantic HTML5 and custom CSS3, featuring a mobile-first responsive layout.

🌐 **Live Demo:** [positivus-ui-design.vercel.app](https://positivus-ui-design.vercel.app/)

---

## 🚀 Features
- **Semantic HTML5:** Clean and SEO-friendly structural markup.
- **Custom CSS3:** Modern styling with robust layout techniques.
- **Fully Responsive:** Optimized for all screen sizes (Desktops, Tablets, and Mobile phones) using explicit CSS Media Queries.
- **Clean UI/UX:** High-fidelity replication of the professional Positivus agency design layout.

## 🛠️ Tech Stack
- **HTML5** - Structure and Content
- **CSS3** - Custom Styling, Layouts, and Typography
- **Media Queries** - Mobile-first Responsive Design
